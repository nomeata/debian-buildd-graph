What ist his?

This plots the number and build times of uploads to the Debian buildd database on architecture , with special emphasis your . The statistics below combine the data from the selected range; you can select ranges by dragging in the graphs, or by using the buttons below.

This was created by Joachim Breitner and the source can be obtained via git. Improvements are welcome, especially with regard to the HTML styling.
